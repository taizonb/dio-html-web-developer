Entendendo Comunicação Client x Server

A internet foi desenvolvida por Universidades dos EUA e as Forças Armadas dos EUA a expandiram durante o período da Guerra Fria. Com o tempo, mais Universidades e outras instituições aderiram ao projeto, até chegar ao tamanho que ela é hoje.

Conforme expandiu e popularizou-se, empresas particulares criaram servidores e cabearam os países para que todos pudessem ter o acesso a rede.

Desta forma, os conceitos tiveram que sser aprimorados, com a diferenciação entre Server-Side e Client-Side, Linguagens de Programação (lembrando que HTML é uma linguagem de marcação).

Introdução ao HTML na Prática

Para utilizar o HTML, podemos contar com a ajudar de editores de código para programadores. Assim facilitando na geração de conteúdo, testes e apresentação.

Em HTML5, utilizamos as chamadas "TAG's" para fazer a marcalçao de conteúdos, e os navegadores conseguem compilar os códigos e apresentar de uma maneira visualmente mais atrativa para qualquer pessoa.
